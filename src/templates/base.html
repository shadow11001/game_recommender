Option 2: Copy/Paste Data (Manual)

If the above link only gave you IDs (no playtime), try this:

  1. Go to your GOG Profile Games Page.
  2. Open DevTools (F12) -> Network tab -> Select Fetch/XHR filter.
  3. Refresh the page.
  4. Find a request named games or stats.
  5. Copy its Response JSON and paste it here.
